# Re-generating the PLIC

rv_plic has been autogenerated from the templated version in the OpenTitan
repository for the commit used by the vendored in OpenTitan IP
(https://github.com/lowRISC/opentitan/tree/16bc8ecfaeca634081044eb43ea8975fd2901768/hw/ip_templates/rv_plic).
It can be re-generated using the `ipgen.py` tool in the OpenTitan repository.
With $OT_ROOT set to an OpenTitan repository (with commit
16bc8ecfaeca634081044eb43ea8975fd2901768 checked out), run the following
command to re-generate:

```sh
$OT_ROOT/util/ipgen.py generate -C $OT_ROOT/hw/ip_templates/rv_plic/ -c ./data/rv_plic_cfg.hjson -o ./rtl/system/autogen/rv_plic
```

This will require the directory this README is contained it to be removed. This
README is not recreated and will need recreating (replacing the generated
README). The generated `doc/` should also be removed.

Note the auto-generated code requires manual modification for use in Sonata. The
changes needed can be seen in `vendor/patches/lowrisc_ip/rv_plic`. It is
unlikely these patches will apply directly (they are specific to the OpenTitan
earlgrey rv_plic) but it should be simple to work out how to do similar changes
in the Sonata specific PLIC.
